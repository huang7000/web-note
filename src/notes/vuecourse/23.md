:::snippet 渲染函数
渲染函数，它比模板更接近编译器。

```html
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
```

```javascript
Vue.component('anchored-heading', {
  render: function (createElement) {
    return createElement(
      'h' + this.level,   // 标签名称
      this.$slots.default // 子节点数组
    )
  },
  props: {
    level: {
      type: Number,
      required: true
    }
  }
})
```
