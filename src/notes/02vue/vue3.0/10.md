:::snippet 基本示例

1. 组件是可复用的组件实例，所以它们与 `new Vue` 接收相同的选项，例如 `data`、`computed`、`watch`、`methods` 以及生命周期钩子等。仅有的例外是像 `el` 这样根实例特有的选项。

```html
<script src="https://unpkg.com/vue@next"></script>
<div id="app">
  <button-counter></button-counter>
  <button-counter></button-counter>
  <button-counter></button-counter>
</div>
```

```javascript
const app = Vue.createApp({});
app.component("button-counter", {
  data() {
    return {
      count: 0,
    };
  },
  template: `
    <p>
      <button @click="count++">
        You clicked me {{ count }} times.
      </button>
    </p>
    `,
});
app.mount("#app");
```


```javascript
const app = Vue.createApp({
  data() {

      posts: [

          id: 1,

        <div>

```

:::snippet 监听子组件事件

1. 我们可以调用内建的`$emit`方法并传入事件的名字，来向父级组件触发一个事件：
2. 使用`$emit`的第二个参数来提供这个值,在父级组件监听这个事件的时候，我们可以通过`$event`访问到被抛出的这个值

```html
<script src="https://unpkg.com/vue@next"></script>
<div id="app" class="demos">
  <blog-post
    :style="{ fontSize: postFontSize + 'em',width: widthSize+'px', height: heightSize+'px',
                background: colorValue}"
    v-bind:key="post.id"
    v-bind:post="post"
    v-on:parent-font="postFontSize += 0.2;"
    v-on:parent-style="widthSize += $event;"
    v-on:parent-styles="widthSize += $event[0];heightSize += $event[1];"
    v-on:parent-style-fun="parentStyleFun"
  >
  </blog-post>
</div>
```

```javascript
const app = Vue.createApp({
  data() {
    return {
      post: {
        id: 1,
        title: "博客Vue",
        content: "vue 学习笔记",
      },
      postFontSize: 1,
      widthSize: 400,
      heightSize: 160,
      colorValue: "red",
    };
  },
  methods: {
    parentStyleFun: function(enlargeAmount, enlargeAmount1) {
      this.heightSize += enlargeAmount;
      this.widthSize += enlargeAmount1;
    },
  },
});
app.component("blog-post", {
  props: ["post"],
  template: `
    <div class="blog-post">
      <h3>{{ post.title }}</h3>
      <button v-on:click="$emit('parent-font')">
        字体
      </button>
      <button v-on:click="$emit('parent-style',10)">
       一个参数  
      </button>
      <button v-on:click="$emit('parent-styles',[10,20])">
       数组参数
      </button>
      <button v-on:click="$emit('parent-style-fun',50,20)">
       方法
      </button>
      <div v-html="post.content"></div>
    </div>
 `,
});
app.mount("#app");
```

:::snippet 自定义事件也可以用于创建支持`v-model`的自定义输入组件

```html
<input v-model="searchText" />
等价于
<input :value="searchText" @input="searchText = $event.target.value" />
```

```html
<script src="https://unpkg.com/vue@next"></script>
<div id="app">
  <h2>customText</h2>
  <custom-input v-model="customText"></custom-input>
  {{customText}}
  <h2>customText2</h2>
  <custom-input
    :model-value="customText2"
    @update:model-value="customText2 = $event"
  ></custom-input>
  {{customText2}}
  <h2>computedText 计算属性重构</h2>
  <computed-input v-model="computedText"></computed-input>
  {{computedText}}
  <h2>inputText</h2>
  <input v-model="inputText" />
  {{inputText}}
</div>
```

```javascript
const app = Vue.createApp({
  data() {
    return {
      inputText: "",
      customText: "",
      customText2: "",
      computedText: "",
      computedText2: "",
    };
  },
  methods: {},
});
app.component("custom-input", {
  props: ["modelValue"],
  template: `
    <input
      :value="modelValue"
      @input="$emit('update:modelValue', $event.target.value)"
    >
  `,
});
app.component("computed-input", {
  props: ["modelValue"],
  template: `
    <input v-model="value">
  `,
  computed: {
    value: {
      get() {
        return this.modelValue;
      },
      set(value) {
        this.$emit("update:modelValue", value);
      },
    },
  },
});
app.mount("#app");
```

:::snippet 动态组件

```html
<script src="https://unpkg.com/vue@next"></script>
<div id="app" class="demo">
  <button
    v-for="tab in tabs"
    v-bind:key="tab"
    v-bind:class="['tab-button', { active: currentTab === tab }]"
    v-on:click="currentTab = tab"
  >
    {{ tab }}
  </button>
  <component v-bind:is="currentTabComponent" class="tab"></component>
</div>
```

```javascript
const app = Vue.createApp({
  data() {
    return {
      currentTab: "Home",
      tabs: ["Home", "Posts", "Archive"],
    };
  },
  computed: {
    currentTabComponent() {
      return "tab-" + this.currentTab.toLowerCase();
    },
  },
});

app.component("tab-home", {
  template: `<div class="demo-tab">Home component</div>`,
});
app.component("tab-posts", {
  template: `<div class="demo-tab">Posts component</div>`,
});
app.component("tab-archive", {
  template: `<div class="demo-tab">Archive component</div>`,
});
app.mount("#app");
```

:::snippet 解析`DOM`模板时的注意事项

> `<ul>`、`<ol>`、`<table>`和`<select>`，对于哪些元素可以出现在其内部是有严格限制的。而有些元素，诸如`<li>`、`<tr>`和`<option>`，只能出现在其它某些特定的元素内部。<br>
> 如果我们从以下来源使用模板的话，这条限制是不存在的：字符串 (例如：`template: '...'`)、单文件组件 (`.vue`)、`<script type="text/x-template">`

```html
<script src="https://unpkg.com/vue@next"></script>
<div id="component-row">
  错误
  <table>
    <blog-post-row></blog-post-row>
  </table>
  正确
  <table>
    <tr is="blog-post-row"></tr>
  </table>
</div>
```
